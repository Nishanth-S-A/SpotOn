How it works

Simple setup,
instant music IQ.

SpotOn runs in Socket Mode — no public URL or server needed.

01
Create a Slack App

Set up a Slack app in Socket Mode, register your slash commands, and grab your Bot & App tokens.

02
Add your tokens

Drop SLACK_BOT_TOKEN and SLACK_APP_TOKEN into a .env file. No other keys required.

03
Run the bot

Install dependencies with npm install and start with node index.js. That's it.

04
Use in Slack

Invite SpotOn to any channel and start firing slash commands. Responses appear instantly, formatted for readability.
